# Resistencia Natural

Proyecto Final – INFORMATORIO 2025  
Desarrollo Web con Python y Django

---

## Descripción del proyecto

**Resistencia Natural** es una aplicación web desarrollada con el framework **Django**, cuyo objetivo es difundir información turística y cultural de la ciudad de Resistencia, incluyendo secciones como artículos, comentarios y gestión de usuarios.

El proyecto fue realizado como **Trabajo Práctico Final** del INFORMATORIO, cumpliendo con los requerimientos de desarrollo, versionado y publicación.

---

## Tecnologías utilizadas

- Python
- Django
- SQLite3
- HTML / CSS
- Git y GitHub
- Render (deploy)

---

## Aclaración importante sobre archivos versionados

De acuerdo con las buenas prácticas de desarrollo, existen archivos que **normalmente no deberían subirse a un repositorio Git**, tales como:

- Base de datos (`db.sqlite3`)
- Archivos compilados de Python (`__pycache__`, `.pyc`)
- Archivos sensibles de configuración local

Sin embargo, **para este trabajo práctico final**, y **con fines exclusivamente académicos y de evaluación**, se decidió **incluir temporalmente** los siguientes archivos:

- `db.sqlite3`  
  - Contiene usuarios, superusuario y artículos de ejemplo necesarios para la correcta visualización y evaluación del proyecto.
- Archivos generados automáticamente por Django en entorno local.

**Esta decisión se tomó únicamente para facilitar la corrección del proyecto**, permitiendo a los evaluadores acceder a un sistema funcional con datos precargados.
